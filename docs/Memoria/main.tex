\section{¿Qué es ELCO-Dealer?}
El \textbf{ELCO-Dealer} es el prototipo desarrollado en este proyecto, concebido como un sistema autónomo y modular capaz de barajar y repartir cartas de forma precisa, repetitiva y controlada. Su finalidad es automatizar completamente el proceso de reparto, reduciendo la intervención humana y garantizando la consistencia del sistema.

El diseño global se estructura en tres subsistemas principales: mecánico, electrónico y de control, que trabajan de forma coordinada.

\begin{figure}[H]
    \centering
    \includegraphics[width=\textwidth]{BARAJADOR (1).png}
    \caption{Primer diagrama del planteamiento del sistema}
    \label{fig:diagrama}
\end{figure}

\subsection{Subsistema mecánico (estructura y movimiento)}

El subsistema mecánico está formado por una estructura diseñada e impresa en 3D que integra tres elementos principales: los rodillos de arrastre, el soporte del mazo de cartas y un plato giratorio encargado del reparto.

El diseño mecánico ha sido optimizado para asegurar una correcta manipulación de las cartas, garantizando su separación individual y reduciendo la probabilidad de atascos durante el funcionamiento continuo del sistema.

\subsection{Subsistema electrónico (sensado y potencia)}

El sistema electrónico se basa en una \textbf{Arduino MEGA 2560 R3}, que actúa como unidad central de control encargada de coordinar todos los procesos.

Los actuadores del sistema (motores DC) son gestionados mediante el controlador \textbf{L298N}, que permite regular la corriente y dirección de los mismos de forma eficiente.

El sistema de sensorización incluye una \textbf{barrera infrarroja}, que permite detectar el paso de cada carta y asegurar la correcta ejecución del reparto bajo el principio de una carta por detección. Además, un \textbf{sensor HOME} proporciona una referencia de posición inicial, permitiendo la calibración del sistema en cada arranque.

Por otro lado, el uso de un \textbf{Slip Ring} permite la transmisión eléctrica continua en la parte giratoria del sistema, posibilitando rotaciones de 360º sin limitaciones por cableado.

\subsection{Subsistema de control (interfaz y lógica)}

El subsistema de control integra tanto el software embebido como la interfaz de usuario. A través de una \textbf{pantalla LCD} y un conjunto de pulsadores, el usuario puede configurar parámetros como el número de jugadores o el modo de funcionamiento de la máquina de forma intuitiva.

El programa implementado en Arduino coordina todos los elementos del sistema, sincronizando sensores, actuadores y estados de operación para garantizar un funcionamiento estable y determinista.

\subsection{Escalabilidad del sistema}

Aunque se trata de un prototipo funcional, el diseño del ELCO-Dealer ha sido planteado con una visión de escalabilidad hacia un posible producto de consumo, contemplando mejoras como la reducción de costes, la optimización de componentes y la transición a placas electrónicas personalizadas (PCBs), con el objetivo de facilitar una futura producción a mayor escala.
