# Documentación

Estructura lógica de documentos del proyecto ELCO-Dealer:

- `Memoria/`: memoria principal y assets asociados.
- `Manual_usuario/`: fuentes LaTeX del manual de usuario.
- `Guia_software/`: fuentes LaTeX de la guía de software.
- `_dist/`: PDFs generados listos para publicar o enlazar desde la web.
- `_build/`: artefactos temporales de compilación LaTeX (`.aux`, `.log`, `.toc`, etc.).
- `bringup_checklist.md`: checklist operativo para puesta en marcha física.

Para editar contenido, trabajar sobre las carpetas de fuentes (`Manual_usuario`,
`Guia_software` y `Memoria`). Los archivos de `_build` no son documentación
editable.
