Este manual de usuario explica el uso y montaje del prototipo \textbf{ELCO-Dealer}, desarrollado como proyecto de la asignatura \textbf{Electrónica de Consumo} del Grado en Ingeniería de Tecnologías y Servicios de Telecomunicación de la Universidad Politécnica de Madrid.

El documento tiene como objetivo orientar a un usario que pretenda montar   y/o usar el prototipo {ELCO-Dealer}. Para hacer esto se describen las dos partes principales del prototipo: El mecanismo de barajado y el mecanismo de reparto. 

El montaje del prototipo está disponible en YouTube (enlace).
